      ✏️ Writing Set-Builder

      General form:
      { variable ∈ domain | condition }

      There is often more than one correct way to write a set in set-builder notation. Both symbolic and English descriptions are valid!

      📝 Examples
      {2,4,6,8} { x ∈ ℕ | x is even, x ≤ 8 }
      {1,4,9,16} { x ∈ ℕ | x = n² for n ∈ ℕ, n ≤ 4 }
      {−2,−1,0,1,2} { x ∈ ℤ | |x| ≤ 2 }
      🔑 Common Patterns
      Evenx = 2k, k ∈ ℕ
      or "x is even"
      Oddx = 2k−1, k ∈ ℕ
      or "x is odd"
      Multiplesx = kn for k ∈ ℕ
      Rangea ≤ x ≤ b
      Squaresx = k², k ∈ ℕ
